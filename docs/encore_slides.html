01 · THE PROBLEM

游戏 LIVE 的观众,
是被动消费的。

主播打出 ace、抢龙、决赛圈最后一击 —— 观众只能看,然后刷条弹幕。 高光发生的那 5 秒,观众的注意力是峰值的,但他们没有任何动作可以"参与"。 礼物是单向,弹幕是文字。观众的高光时刻情绪没被接住,转化为关注和回访的概率被白白浪费。

02 · WHAT IS ENCORE

把主播的高光时刻,实时重构成观众能玩的 30 秒迷你游戏。

STEP 01
👁

OBSERVE

AI Vision 持续观察 LIVE 画面,识别 kill / clutch / ace / 抢龙 等高光事件。本质是把"主播按高光按钮"自动化。

STEP 02
🧬

STRUCTURE

把高光时刻提取成一份 JSON config:游戏类型、地图主题、敌人数、起始 HP、武器、场景描述。这就是 V2G 的"语言"。

STEP 03
🎮

RECONSTRUCT

客户端在 30 秒内把 config 渲染成一个体素像素风的 mini game,以 bottom sheet 形式弹给当时在场的观众。

07 · V2G ARCHITECTURE

观察 → 推理 → 配置 → 播放,全链路 ≤ 6 秒。

┌─────────────────────────────────────────────────────────────────┐
│  LIVE Source      →   Frame Sampler     →     Claude Vision   │
│  (TikTok / video)        (every 4s, 480px JPEG)    (returns JSON)  │
│                                  ↓                              │
│                ┌─────────────────────────────┐                   │
│                │  V2G Config JSON            │                   │
│                │  template: fps                │                   │
│                │  theme:    desert             │                   │
│                │  scenario: enemy_count: 3   │                   │
│                │            hp_start:    55  │                   │
│                │            weapon: rifle    │                   │
│                │            desc: "1v3 retake"│                   │
│                └─────────────────────────────┘                   │
│                                  ↓                              │
│  Streamer page   ─→   postMessage   ─→  iframe (game)      │
│  ▸ video player        {type:'launch',         ▸ apply theme      │
│  ▸ status overlay       config: {...}}         ▸ apply scenario   │
│  ▸ sheet animation                             ▸ startGame()     │
│                                                ↓  30s play       │
│  Streamer page   ←─   postMessage   ←─   iframe (game)      │
│  ▸ slide sheet down    {type:'encore_done',    ▸ game over        │
│  ▸ cooldown 8s          stats: {won, kills}}   ▸ post stats       │
│  ▸ resume sampling                                                │
└─────────────────────────────────────────────────────────────────┘
